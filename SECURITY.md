# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| > 0.4.x | :white_check_mark: |
| < 0.4   | :x:                |

## Reporting a Vulnerability

Please report a vulnerability by using [Github Issues](https://github.com/MasterLaplace/Flakkari/issues/new/choose).
