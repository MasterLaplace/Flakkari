std::vector< std::pair< std::shared_ptr< Address >, Buffer > > Flakkari::Network::Socket::receiveBatch ( uint32_t  maxMessages = 16u,
int  flags = 0 
) const

Receive multiple UDP messages in one syscall (batch). This function is only used by UDP sockets.

Parameters
maxMessagesMaximum number of messages to receive in one call.
flagsFlags to pass to recvfrom / recvmmsg.
Returns
vector of (Address, Buffer) pairs received. Empty if none or on EAGAIN/EWOULDBLOCK.
Note
On Linux this uses recvmmsg for efficiency. On other platforms it falls back to repeated recvfrom.
